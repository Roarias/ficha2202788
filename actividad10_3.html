<audio< HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus paginas, Al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src: Este atributo especifica la runta de acceso al archivo de audio

controls: Este atributo indica si el reproductor debe mosntrar los controles. Si no utiliza este atributo, no se mostraran los contrloes de forma predeterminada. Tmabien puede especificar sus propios controles mendiante javascript

autoplay: La precensia de este atributo indica que el audio debe empezar a reproducirse automatiamente. (Se comside una mejor practica permitir al usario que reprodusca el audio)

loop: Este atributo especifica que las pista de audio debe reproducirse de nuevo una ves que haya finalizado

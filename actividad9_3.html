El elemento <div< permite agrupar un conjunto de lelemtos juntos en un cuadrado nivel de bloque. Por ejemplo: puede crear elementos <div< para que contega todos los elementos del encabezado del sitio (el logotipo y navegacion), o puede crear un elemento <div< para que contenga infomacion.

En un navegador, el contenido del elemento <div< comenzara en una linea, pero aparte de esto no habra ninguna difenrecia en la presentacion de la pagina. Sin embargo, el uso de un atributo ID o class en el elemento <div< significa que puede crear reglas de estilos CSS para indicar cuanto espacio debe ocupar el elemento <div< en la pantalla y cambiar la apariencia de todos los elementos contenidos en el.Tambien puede facilitar el seguimiento del codigo si ha utilizado elementos <div< para contener cada seccion de la pagina.

Dado que puede haber varios otros elementos dentro un elemento <div<, puede ser util agregar un comentario despues de la etiqueta </div< de cierre. Esto le permite ver claramente a que la etiqueta de apertura es supone que conrresponde.

La etiqueta video: Para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro de elemento <video>, (Esto dee reemplezar el atributo src en la etiqueta <video> de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el esta disponible en diferentes formatos. (Debido a un error el ipad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: Este atributo especifica la runa de acceso al video

type: Se debe ulitizar este atributo para decirle al navagador que formato es el video. De lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda)

codecs: El codec que uso para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, asi como comillas dobles en el atributo de type, cuando se suministra
